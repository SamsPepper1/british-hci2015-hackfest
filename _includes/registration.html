How to Register for the Hackfest

email - raise issue - pull request

email - the easiest way!!

The link above with populate an email appropriately, but if you prefer the manual approach read below

Email: bhci2015hackfest@gmail.com with the following details:

  • name;
  • tagline;
  • institution;
  • institution url;
  • institution logo;
  • personal website;
  • biography (100-200 words)

and optional some social media urls.

  • github username;
  • facebook id;
  • twitter handle;
  • linkedin id

any other notes, such as, "I'll only be coming for the Monday as I'm involved in the Doctoral Networking Day (Tuesday, July 14th 2015)"
